## Project Description

📱 **Revolutionary Shopping App** 📱

Our mobile app is designed to offer a seamless and enjoyable shopping experience for a wide variety of products (excluding food). 

### Key Features:
- 🔒 **Login**: Supports custom credentials .
- ✨ **Registration**: Easy sign-up process.
- 🏠 **Home Screen**: Welcomes users by name and displays an extensive inventory of products, organized intuitively for easy browsing.
- 🎯 **Personalized Experience**: 
  - Customize preferences (language, dark mode)
  - Manage a wishlist
- 🛒 **Shopping Cart**: 
  - Review selections
  - Calculate costs
- 🔍 **Product Details**: Dedicated pages with detailed information, reviews, and ratings.
- ☁️ **Infrastructure**: Built on Firebase Cloud for reliable and scalable service.
